## Description

  A Node.js server-side application framework for building efficient and scalable [GNews](https://gnews.io/) APIs with caching and limits
  
## Installation

  

```bash

$ yarn  install

```

  

## Running the app

  

```bash

# development

$ yarn  run  start

  

# watch mode

$ yarn  run  start:dev

  

# production mode

$ yarn  run  start:prod

```

  

## Test

  

```bash

# unit tests

$ yarn  run  test


# test coverage

$ yarn  run  test:cov

```
